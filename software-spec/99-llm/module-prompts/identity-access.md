# Промпт модуля доступа и идентичности

Реализуй владельца правил идентичности внутри server-rendered Rails-монолита.
Прочитай весь `01-modules/identity-access/`, HTTP-001—HTTP-020,
HTTP-041—HTTP-043, ADR-003, домен, данные и интеграции. Сначала создай tests
Rails forms/views/cookies и WebAuthn protocol, затем e-mail, sessions,
Telegram и безопасный merge.

# Таблицы и коллекции

| Таблица | Кратность/роль | Удаление |
|---|---|---|
| `WebUser` | корень агрегата | пользовательского удаления нет |
| `WebSession` | много на пользователя | каскад с пользователем |
| `WebRefreshToken` | история использованных refresh | каскад с сеансом |
| `WebAuthnCredential` | ключи доступа | каскад с пользователем |
| `WebAuthnChallenge` | одноразовые церемонии | каскад с пользователем, если он задан |
| `TelegramAuthState` | OIDC state/nonce/PKCE | при удалении пользователя ссылка обнуляется |
| `EmailVerificationCode` | коды и попытки | каскад с пользователем |
| `AccountMergeConfirmation` | явное объединение | каскад с пользователем |
| `PaymentOperation` | долговечная команда | запрещает удаление пользователя |
| `PaymentRecord` | внешняя платёжная проекция | каскад с пользователем; ссылка на операцию обнуляется |
| `PaymentHistorySyncState` | одна строка на пользователя | каскад с пользователем |
| `AuditLog` | добавляемые эксплуатационные факты | ссылка на пользователя обнуляется |
| `RateLimitEvent` | антизлоупотребление | независима |
| `AppSetting` | JSON-настройка по ключу | независима |
| `IntegrationStatus` | один снимок на сервис | независима |

Полный состав столбцов задаёт `field-catalog.md`, ограничения — `constraints.md`, связи — `03-domain/relationships.md`.

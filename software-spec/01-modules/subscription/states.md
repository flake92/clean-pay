# Состояния

## Подписка

- отсутствует;
- существует и активна;
- существует с иным внешним состоянием;
- существует, но актуальная ссылка временно недоступна.

Точное текстовое состояние приходит извне и сохраняется без изменения регистра. Интерфейс знает подписи для известных значений и показывает неизвестное значение дословно.

## Ссылка доступа

`не запрошена → найдена и проверена` либо `не найдена/неоднозначна/неактивна`. Только проверенное состояние позволяет показать кнопку подключения.

## Устройства

`загрузка → пустой список | список устройств | ошибка`; отдельное устройство может находиться в состоянии удаления, а удаление всех блокирует общий элемент управления.

## Промокод и перевыпуск

`готово → выполняется → успешно | отклонено | внешняя ошибка`.

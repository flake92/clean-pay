# Исходящие интерфейсы

- Remnashop: тарифы, предложения, подписка, устройства, перевыпуск и промокод.
- Remnawave: поиск фактической ссылки по идентификатору пользователя, почте и Telegram.
- Модуль доступа: получение подтверждённого пользователя и внешней сессии.
- Модуль платежей: предоставление актуального предложения для проверки команды.
- Аудит: успешные и неуспешные изменяющие действия.

Контракты внешних систем: `04-integrations/remnashop.md` и `04-integrations/remnawave.md`.

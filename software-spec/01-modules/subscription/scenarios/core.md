# Сценарии

## Загрузка кабинета

1. Получить текущую подписку.
2. Если она существует, найти и проверить фактическую ссылку.
3. Параллельно получить устройства, предложения, историю платежей и контакты поддержки.
4. Независимо от ошибок второстепенных секций показать доступные данные каждой секции.

## Управление устройством

1. Пользователь выбирает одно устройство или удаление всех.
2. Соответствующая кнопка блокируется.
3. После успешной внешней операции список перечитывается либо обновляется.
4. Ошибка показывается без ложного удаления элемента.

## Перевыпуск и промокод

Изменяющая команда отправляется один раз на нажатие, кнопка блокируется, результат отображается пользователю и фиксируется в аудите. После перевыпуска необходимо заново получить актуальную ссылку.

# Пользовательский интерфейс модуля

Основные поверхности: карточки тарифов, метрики кабинета, карточка подключения, подробности подписки, таблица/карточки устройств, форма промокода и действия «Подключиться», «Скопировать ссылку», «Перевыпустить подписку», «Удалить», «Удалить все устройства», «Выбрать тариф», «Изменить тариф», «Продлить».

Интерфейс должен отдельно показывать загрузку, отсутствие подписки, отсутствие устройств, частичную ошибку секции, недоступную ссылку и выполняемое действие.

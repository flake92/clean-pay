# Ответственность

- выдавать общедоступный каталог тарифов;
- выдавать персональные предложения с ценами и способами оплаты;
- получать состояние текущей подписки;
- заменять сохранённую внешнюю ссылку фактической актуальной ссылкой доступа;
- выдавать и изменять список подключённых устройств;
- перевыпускать ссылку доступа;
- активировать промокод;
- предоставлять платёжному модулю актуальное предложение для проверки цены.

Модуль не проводит оплату и не определяет окончательный результат платежа.

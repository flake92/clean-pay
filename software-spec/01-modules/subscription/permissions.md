# Права доступа

Публичный каталог доступен гостю. Остальные операции разрешены только полной сессии, прошедшей правило подтверждения почты и имеющей доказанную внешнюю учётную запись. Ограниченная сессия, неподтверждённый пользователь без Telegram и внутренний исполнитель без пользовательской сессии доступа не получают.

# Модуль доступа и идентичности

Модуль устанавливает, кто является пользователем, каким способом он подтвердил вход и какие действия ему разрешены. Он объединяет вход по почте, Telegram и ключу доступа в одного владельца, но не принимает решений о тарифе или результате платежа.

Входы: HTTP-001—HTTP-020, HTTP-039, HTTP-041—HTTP-043. Выходы: пользовательская сессия, профиль, cookie, перенаправление, запросы к Remnashop/Telegram/Turnstile и изменения данных идентичности.

# Входные интерфейсы

| Группа | Операции |
|---|---|
| Почтовый вход | HTTP-001—HTTP-009 |
| Ключи доступа | HTTP-010—HTTP-015 |
| Telegram и объединение | HTTP-016—HTTP-019, HTTP-041—HTTP-043 |
| Связывание внешней учётной записи | HTTP-020 |

Точные транспортные контракты находятся в `02-interfaces/http/identity.md`. Каждый интерфейс входит в этот модуль через самостоятельную логическую операцию; похожие маршруты не объединяются.

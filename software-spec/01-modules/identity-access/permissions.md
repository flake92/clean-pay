# Права доступа

| Состояние | Разрешено |
|---|---|
| Гость | определение адреса, регистрация, почтовый вход, начало/возврат Telegram, вход по ключу доступа, публичный выход |
| Ограниченная сессия | создание ключа доступа, завершение его регистрации, выход |
| Полная сессия с неподтверждённой почтой и без Telegram | операции подтверждения/изменения почты и выход |
| Полная сессия с подтверждённой почтой или Telegram | профиль, пароль, ключи доступа, связывание и все персональные возможности других модулей |
| Внутренний исполнитель | доступа к пользовательским операциям по служебному секрету не получает |

Наличие пункта меню не является проверкой прав: право проверяется на каждом защищённом входе.

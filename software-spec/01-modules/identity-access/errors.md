# Ошибки модуля

| Категория | Примеры реакции |
|---|---|
| Некорректный ввод | 400 с точным публичным кодом проверки данных |
| Неверные данные входа | 401 без раскрытия существования пароля |
| Нет или истекла сессия | 401 и удаление недействительных cookie там, где это делает пограничная политика |
| Недостаточный уровень/неподтверждённая почта | 403 либо перенаправление на обязательный экран |
| Ресурс не принадлежит пользователю | 404 без раскрытия чужого объекта |
| Совпадение владельцев требует решения | 409 и переход к объединению |
| Одновременное изменение | 409 либо безопасный повтор чтения/захвата |
| Слишком много попыток | 429 |
| Внешняя система недоступна | 502/503/504 согласно общему контракту ошибок |
| Неожиданная ошибка | 500 без внутренних подробностей в рабочем окружении |

Полный список публичных кодов и сообщений находится в `02-interfaces/error-contracts.md`.

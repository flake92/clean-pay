# Предметные понятия

- **Пользователь** — итоговый владелец всех способов входа и данных.
- **Способ входа** — почта, Telegram либо ключ доступа.
- **Сессия** — разрешение браузеру действовать от имени пользователя до отзыва или истечения.
- **Уровень сессии** — полный либо ограниченный.
- **Одноразовый вызов** — временная запись для регистрации/входа по ключу доступа.
- **Состояние Telegram-входа** — временная связь между началом и возвратом из Telegram.
- **Ожидающий адрес** — новый адрес до подтверждения кодом.
- **Свидетельство объединения** — данные, доказывающие, какие внешние учётные записи должны стать одним пользователем.
- **Подтверждение объединения** — ограниченное по времени явное решение пользователя.

# Исходящие интерфейсы

- Remnashop: регистрация, проверка пароля, профиль, обновление внешней сессии, изменение пароля/почты, подтверждение почты, Telegram-связывание и объединение внешних владельцев.
- Telegram: начало авторизации, обмен кода, проверка подписанного удостоверения.
- Turnstile: проверка пользователя в защищённых публичных сценариях.
- Постоянное хранилище: пользователи, сессии, одноразовые состояния, ключи доступа, подтверждения объединения и аудит.
- Быстрое хранилище: ограничение частоты запросов.
- Браузер: основные и временные cookie, перенаправления, запрос к средству ключа доступа.

Точные контракты внешних систем находятся в `04-integrations/`.

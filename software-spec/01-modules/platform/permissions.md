# Права доступа

Проверки жизни, публичная готовность и служебный сценарий доступны публично. Подробная готовность защищена отдельным секретом. Сведения поддержки по фактической пограничной политике требуют пользовательскую сессию. Совместимый выход публичен, но изменяющий запрос из браузера обязан иметь доверенный источник. Фоновые команды доступны только внутренней сети и своему секрету.

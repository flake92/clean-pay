# Сценарии

## Запуск

Оператор передаёт конфигурацию → система проверяет обязательность, форматы, происхождение адресов, стойкость и различие секретов → применяет ожидающие миграции → начинает обслуживание → оркестратор ждёт подробной готовности.

## Проверка готовности

Внутренний запрос запускает параллельные ограниченные проверки зависимостей → сохраняет снимок → возвращает подробности. Публичный запрос читает только свежий агрегированный снимок.

## Очистка

Исполнитель вычисляет границы сроков → удаляет только подходящие одноразовые состояния, старые сессии, аудит и события ограничения частоты → журналирует количества → обновляет сердцебиение.

## Установка приложения

При доступном системном приглашении пользователь подтверждает установку. На iOS и во встроенных браузерах показывается соответствующая инструкция. В установленном состоянии кнопка открывает кабинет. Без сети навигация получает автономную страницу.

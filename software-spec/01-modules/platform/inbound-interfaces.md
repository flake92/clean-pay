# Входные интерфейсы

- HTTP-033: сведения поддержки;
- HTTP-034 и HTTP-035: простая проверка жизни;
- HTTP-036: безопасная публичная готовность;
- HTTP-037: защищённая подробная готовность;
- HTTP-040: совместимый выход;
- HTTP-044: служебный сценарий устанавливаемого приложения;
- переменные окружения и параметры запуска;
- сигналы оркестратора и периодические тики фоновых исполнителей.

# Исходящие интерфейсы

- проверки БД, быстрого хранилища, Remnashop, Telegram и необязательных Mailpit/Remnawave;
- внутренний вызов команды сверки платежей;
- удаление устаревших служебных записей;
- структурированный вывод журналов;
- состояние здоровья контейнера и файлы сердцебиения;
- браузерный кэш общедоступной оболочки и автономной страницы;
- миграции постоянного хранилища и операции резервирования/восстановления оператора.

# Пользовательский интерфейс модуля

Модуль предоставляет общую верхнюю панель, боковое меню, оболочку страниц, главную страницу, поддержку, установку и автономный экран. Меню меняется по профилю и наличию предложения продления. На мобильном устройстве боковая и верхняя навигация раскрываются кнопками.

Диалоги установки имеют явные заголовки, признак модальности, кнопку закрытия и пошаговую инструкцию. Автономный экран не обещает работу подписки или платежей без сети.

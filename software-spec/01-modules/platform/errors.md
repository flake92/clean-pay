# Ошибки

- неверная конфигурация или опасное значение — отказ запуска с точным именем настройки;
- деградация зависимости — 503 готовности, но не обязательно остановка процесса;
- неверный служебный секрет — 404;
- ошибка подробной проверки — безопасный сокращённый 503;
- ошибка фоновой команды — запись в журнал и повтор на следующем интервале;
- отсутствие сети в браузере — автономная страница только для навигации;
- ошибка совместимого выхода может дать обычный 500, так как старый контракт не использует стандартную оболочку ошибок.

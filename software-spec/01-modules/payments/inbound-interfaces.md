# Входные интерфейсы

| Операция | Назначение |
|---|---|
| HTTP-024 | начать покупку новой/изменённой подписки |
| HTTP-025 | начать продление |
| HTTP-031 | синхронизировать и получить последние платежи |
| HTTP-032 | определить состояние конкретного или последнего платежа/операции |
| HTTP-038 | выполнить ограниченную фоновую сверку и продолжить историю |

Точные поля, статусы и заголовки: `02-interfaces/http/payments.md`.

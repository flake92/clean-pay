# Предметные понятия

- **Платёжная команда** — покупка или продление с выбранными условиями.
- **Ключ повторяемости** — уникальный идентификатор одной попытки пользователя.
- **Отпечаток запроса** — однозначное представление значимых полей команды.
- **Владелец платежа** — пользователь вместе с доказанной внешней учётной записью.
- **Захват операции** — временное исключительное право одного исполнителя продолжать команду.
- **Отправка** — момент, после которого внешняя система могла создать платёж.
- **Неопределённый исход** — отправка возможна, но достоверного результата ещё нет.
- **Запись платежа** — локальное известное состояние внешней транзакции.
- **Состояние синхронизации истории** — курсор, поколение владельца, попытки и аренда загрузки.

# Проверка живости оркестратора

## Идентификатор

`HTTP-035`

## Назначение

Выделенный liveness probe для решения о перезапуске процесса.

## Владелец

Платформенный модуль.

## Акторы

Оркестратор.

## Предусловия

Процесс принимает HTTP.

## Логический входной контракт

Нет.

## Текущий транспорт

`GET /health/liveness`; публичная машинная JSON liveness-проверка без внешних вызовов.

## Правила валидации

Нет.

## Нормализация

Version как HTTP-034.

## Авторизация

Нет.

## Идемпотентность

Повторяемо.

## Основной сценарий

Немедленный статический ответ без I/O к зависимостям.

## Альтернативные сценарии

Нет.

## Ошибочные сценарии

Только фатальная ошибка процесса/framework.

## Логический результат

`200 application/json` пока Rails-процесс способен обслужить запрос.

## Побочные эффекты

Журнал доступа.

## Транзакционные требования

Нет.

## Наблюдаемость

Нельзя подменять readiness-проверкой: отказ зависимости не должен перезапускать процесс.

## Источники

Доказательства находятся в `09-traceability/`.

## Статус уверенности

`требует повторной проверки после ADR-003`

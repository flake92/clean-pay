# Базовая проверка жизни

## Идентификатор

`HTTP-034`

## Назначение

Подтвердить, что HTTP-процесс способен сформировать ответ, без проверки зависимостей.

## Владелец

Платформенный модуль.

## Акторы

Оркестратор, балансировщик, оператор.

## Предусловия

Процесс принимает HTTP.

## Логический входной контракт

Нет; query/body игнорируются.

## Текущий транспорт

`GET /health`; публичная машинная JSON-проверка состояния приложения.

## Правила валидации

Нет.

## Нормализация

Version берётся из `npm_package_version`, при отсутствии — строка `0.1.0`.

## Авторизация

Нет.

## Идемпотентность

Полностью повторяемо.

## Основной сценарий

Немедленно вернуть статический статус процесса.

## Альтернативные сценарии

Нет.

## Ошибочные сценарии

Только невозможность самого процесса/framework сформировать ответ; handler не возвращает управляемых ошибок.

## Логический результат

`200 application/json` с агрегированным статусом без секретов.

## Побочные эффекты

Журнал доступа.

## Транзакционные требования

Нет.

## Наблюдаемость

Используется как process probe, не как готовность.

## Источники

Доказательства находятся в `09-traceability/`.

## Статус уверенности

`требует повторной проверки после ADR-003`

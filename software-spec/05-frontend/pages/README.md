# Карточки экранов

Этот каталог является нормативным описанием PAGE-001…PAGE-019. Каждая карточка задаёт наблюдаемую структуру, управление, состояния, сетевые операции, адаптивность и критерии приёмки без обращения к файлам старой реализации. Общие цвета, размеры и shell заданы в `../design-tokens.md`; изображения и воспроизводимые сцены — в `../reference-manifest.md`.

Порядок секций в карточке является порядком DOM и визуального чтения сверху вниз. Если секция условна, условие указано явно. Нельзя добавлять диагноз о внутренней реализации: новая технология свободна, но внешний результат и взаимодействия обязательны.

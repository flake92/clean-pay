# Экранные контракты

Полные нормативные карточки находятся в `pages/PAGE-001.md` … `pages/PAGE-019.md`. Для каждого экрана там зафиксированы маршрут и guard, shell, порядок секций, видимые строки и controls, HTTP-операции, состояния, responsive-поведение, accessibility и приёмка. Перечень не зависит от старых путей реализации.

Визуальные доказательства: `reference-manifest.md` и 38 файлов `reference/current/{desktop,mobile}`. Общие значения: `design-tokens.md`. Сквозные переходы: `user-journeys.md`. Матрица UI → API: `api-usage.md`.

Запрещено считать эту сводку заменой PAGE-карточек или эталонов.

# Контрактный макет

`index.html` — автономный кликабельный макет всех 19 текущих маршрутов. Он воспроизводит две оболочки, палитру, геометрию карточек/controls, подписи, навигацию, основные формы/actions и четыре общих состояния (`Данные`, `Загрузка`, `Пусто`, `Ошибка`). Демонстрационные значения нужны только для layout; точные контракты данных находятся в соседних Markdown-документах.

Открытие: любой браузер, файл `mockup/index.html`. В обычном режиме служебные controls скрыты. Для проверки маршрутов и состояний открыть `index.html?controls=1#/cabinet`; URL hash выбирает экран. Эталоны фактического приложения находятся в `reference/current/`, повторный рендер макета — в `reference/mockup/`, точные требования — в `pages/PAGE-*.md`, результаты сверки — в `visual-comparison-report.md`.

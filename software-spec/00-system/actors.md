# Участники

| Участник | Роль |
|---|---|
| Гость | просматривает общедоступные страницы, начинает регистрацию или вход |
| Пользователь с ограниченной сессией | завершает настройку ключа доступа либо выходит |
| Пользователь с неподтверждённой почтой | подтверждает или меняет адрес; остальные возможности ограничены, если нет Telegram |
| Полностью вошедший пользователь | управляет профилем, подпиской, устройствами и платежами |
| Telegram | подтверждает Telegram-идентичность и возвращает пользователя в приложение |
| Remnashop | предоставляет внешнюю учётную запись, тарифы, подписочные и платёжные операции |
| Remnawave | предоставляет фактическую актуальную ссылку доступа |
| Turnstile | подтверждает прохождение защиты от автоматических запросов |
| Платёжный провайдер | принимает оплату после создания платёжной операции |
| Планировщик | периодически запускает внутреннюю сверку и продолжение истории |
| Оператор | развёртывает систему, проверяет готовность, выполняет резервирование и восстановление |
| Тестовая среда | воспроизводит внешние системы без обращения к боевым сервисам |

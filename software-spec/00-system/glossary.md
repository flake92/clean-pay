# Словарь

| Термин | Значение |
|---|---|
| Пользователь | единый локальный владелец способов входа, сессий и платёжных данных |
| Внешняя учётная запись | учётная запись пользователя в Remnashop |
| Полная сессия | сессия, разрешающая обычные защищённые действия |
| Ограниченная сессия | временная сессия, разрешающая только настройку ключа доступа и выход |
| Ключ доступа | средство входа браузера или устройства с подтверждением пользователя |
| Подтверждение объединения | явное согласие пользователя соединить две найденные учётные записи |
| Предложение | сочетание тарифа, длительности, способа оплаты, цены и версии условий |
| Платёжная попытка | одно намерение пользователя начать покупку или продление |
| Платёжная операция | сохраняемая команда, защищающая платёжную попытку от повторного выполнения |
| Запись платежа | локальное представление известного внешнего платежа |
| Неопределённый исход | состояние, когда запрос мог быть выполнен внешней системой, но подтверждения ещё нет |
| Актуальная ссылка доступа | однозначная действующая ссылка, полученная непосредственно из системы выдачи доступа |
| Внешний контракт | наблюдаемые входы, выходы и эффекты системы, которые обязана сохранить новая реализация |

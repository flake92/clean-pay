# Полный реестр исходного дерева

Срез: 2026-07-22. Исключены только `.git`, сгенерированные каталоги и результаты выполнения `node_modules`, `.next`, `.next-mock`, `coverage`, метаданные IDE `.idea` и сам `software-spec`. Этот реестр является контрольным перечнем для матриц трассируемости.

## Полное правило трассировки файлов

Каждая из 340 строк раздела «Файлы» покрыта ровно одним или несколькими правилами ниже; поэтому manifest одновременно служит per-file checklist, а специализированные matrices дают per-element детализацию.

| Path/class | Спецификация |
|---|---|
| `src/app/**/route.ts`, `src/proxy.ts` | `02-interfaces/http/**`, `09-traceability/routes-matrix.md` |
| `src/app/**/page.tsx`, `src/frontend/**`, UI assets/styles | `05-frontend/**`, `frontend-matrix.md`, mockup |
| `src/backend/auth/**`, `sessions/**`, security | identity module, domain, HTTP identity, quality security |
| `src/backend/payments/**` | payments module/cards, domain/data/operations |
| `src/backend/integrations/**` | `04-integrations/**`, integration matrix |
| database/cache/limits/health/observability/config backend | 06-data, 07-operations, configuration, quality |
| `src/shared/**` | compatibility/value objects/HTTP response schemas/frontend flows |
| `prisma/schema.prisma`, all 15 migration files | all 06-data docs + data matrix |
| `deploy/**`, root launchers, Compose/Docker | 07-operations, configuration, backup/quality |
| `.devcontainer/**`, `infra/test/**`, mock scripts | integration mock sections, acceptance strategy |
| `tests/unit/backend/**` | module/card/integration/data rule evidence by tested filename |
| `tests/unit/frontend/**` | frontend matrix/screens/state/API evidence |
| `tests/unit/config/**` | configuration/operations/security/compatibility evidence |
| `tests/integration/**`, `tests/e2e/**` | concurrency/routes/full-stack regression evidence |
| `public/**`, manifest/icons/theme/font | frontend inventory + browser-pwa/files contract |
| `docs/**`, README/license/config/tooling files | source inventory, quality/decision context; code/tests prevail |
| empty `app/infra/test` placeholders | non-runtime inventory only; no behavior to reproduce |
